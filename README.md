# Reproductive Health Power BI Portfolio

## Dashboard: Menstrual Health & Cycle Wellness
Built in Power BI using the Kaggle Menstrual Health Dataset.
Explores cycle patterns, symptoms, lifestyle factors, and hormones 
across 2,000 users and 17,976 cycle logs.

### Pages
- Cycle Health Overview
- Symptoms & Wellbeing
- User Profiles & Lifestyle
- Hormones & Ovulation

### Tools Used
- Power BI Desktop
- Power Query (data cleaning & transformation)
- DAX (calculated measures)

### Data Source
Kaggle Menstrual Health Dataset (Period_Log.csv & User_Profile.csv)
